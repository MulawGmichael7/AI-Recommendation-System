# AI-Based Job Matching & Skill Gap Analysis

AI-based system for job matching and identifying skill gaps using machine
learning and NLP techniques. Matches candidate profiles to job postings
based on skill similarity, identifies the gap between a candidate's
current skills and a target role's requirements, and recommends learning
resources to close that gap.

This project was originally developed as an undergraduate thesis (awarded
A+) at Mekelle University. This repository is a clean, reproducible
rebuild of that pipeline.

## How it works

1. **Job & candidate representation** тАФ each job posting's required
   skills, and each candidate's known skills, are represented as text.
2. **TF-IDF vectorization** тАФ skills are vectorized with
   `TfidfVectorizer` (scikit-learn).
3. **Matching** тАФ a candidate's skill vector is compared against every
   job's skill vector using cosine similarity; the top-k most similar
   jobs are returned.
4. **Skill gap analysis** тАФ for the top-matched job, the system computes
   the set difference between required skills and the candidate's known
   skills.
5. **Learning recommendations** тАФ each missing skill is mapped to a
   suggested learning resource.

## Dataset

`data/jobs.csv` and `data/candidates.csv` are generated by
`data/generate_data.py` тАФ a synthetic but realistic dataset of 15 job
roles and 200 candidate profiles (each candidate generated with a hidden
"true target job" plus some skill noise, which enables honest evaluation
below).

> Public datasets (e.g. scraped job postings from Kaggle/LinkedIn) can be
> swapped in directly тАФ the pipeline only requires a `job_title,
> required_skills` and `candidate_id, skills` CSV structure.

## Results

Evaluated on 200 synthetic candidates against their known true target job
(`src/evaluate.py`):

| Metric | Score |
|---|---|
| Top-1 accuracy | 86.0% |
| Top-3 accuracy | 98.0% |

(Reproduce with `python3 src/evaluate.py`.)

## Running it

```bash

python3 data/generate_data.py   # regenerate the dataset (optional, already included)
python3 main.py                  # run the end-to-end demo
python3 src/evaluate.py          # reproduce the accuracy metrics
```
